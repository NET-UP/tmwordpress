Erstelle und verwalte Veranstaltungen mit TicketMachine Event Manager

== Description ==

Mit TicketMachine Event Manager können Sie leicht Veranstaltungen erstellen und verwalten. Das Plugin stellt viele Funktion bereit um individuelle Lösungen für ihre Homepage zu finden. Das Plugin ist leicht zu verstehen und zu bedienen.

Das Plugin ist so gestaltet, dass Sie es genau auf das Design Ihrer Webseite anpassen können. Die Farbgebung kann genau auf Ihr Farbdesign angepasst werden.
Das Plugin ist auch für die Verwendung auf mobilen Geräten optimiert. Auf dem Smartphone sehen eingerichtete Events immer sehr gut aus und sind leicht zu bedinen.
Alle Veranstaltungen können mit Freunden geteilt werden, so werden Ihre Veranstaltungen noch beliebter. Mit Facebook, Twitter, WhatsApp, E-Mail und Co. können Besucher Ihrer Veranstaltungen ihren Freunden mitteilen wo Sie dieses Wochenende sind.

Sind Sie neu beim TicketMachine Event Manager Plugin? Lesen Sie sich die [Benutzerdokumentation] (Link zu Helpdesk) durch um durchstarten zu können.

= Funktionen =

Hier ist eine Liste der Funktionen des TicketMachine Event Manager Plugins:

### TicketMachine Event Manager Free:
* Veranstaltungen erstellen und verwalten
* Kalender Ansicht können leicht als Widgets via Shortcode eingebunden werden
* Kachel Ansicht kann leicht als Widgets via Shortcode eingebunden werden
* Eventlisten Ansicht können leicht als Widgets via Shortcode eingebunden werden
* Veranstaltungsbilder
* Das Plugin ist SEO-optimiert, ihre Veranstaltungen haben dadurch ein höheres ranking bei Google
* Mehrtagesveranstaltungen
* DSGVO konforme Google Maps integration
* DSGVO konforme Social-Media teilen funktionalität
* individuell Anpassbares Design auf Ihre Webseite
* Übersetzbar mit WPML
* Veranstaltungen sind für mobile Geräte Optimiert
* Suchen / Filtern nach Veranstaltungsname oder Schlagwort
* Outlook Integration (iCal)
* Google Kalender Integration (Gcal)
* BiTV / WCAG konform für uneingeschränkte Benutzung der Veranstaltungen für alle
* W3C valide

== Installation ==

### TicketMachine Eventmanager herunterladen und installieren

### TicketMachine Eventmanager mit Wordpress installieren:

1. Im Wordpress Backend auf "Plugins" navigieren, dort auf "Installieren" klicken.
2. Im Suchfeld nach "TicketMachine Event Manager" suchen.
3. Auf "Jetzt installieren" klicken. 
4. Auf "mit TicketMachine verbinden" klicken, um Ihre Events zuzugreifen. 
5. Wenn gefragt geben Sie ihre E-Mail Adresse ein.
6. Auf "Los geht’s" klicken.

### TicketMachine Eventmanager manuell installieren:

1. Das Plug-In von [LINK ZU WP PLUGIN SEITE] herunterladen
2. Im Wordpress Backend auf "Plugins" navigieren, dort auf "Installieren" klicken.
3. Auf "Plugin hochladen" klicken.
4. Das heruntergeladene ZIP-Datei auswählen und hochladen.
7. Der Anleitung auf dem Bildschirm folgen und warten bis das hochladen erfolgreich ist. 
8. Wenn gefragt geben Sie ihre E-Mail Adresse ein.
9. Auf "Los geht’s" klicken.
